Fscore_instructions

1. What is F-score programme?
F-score programme is assessing strength of company’s financial position by following Piotroski F-Score's Calculation procedure.
This programme is made by python. 


*What is Piotroski F-Score?
	reference : https://en.wikipedia.org/wiki/Piotroski_F-Score
	Piotroski F-Score is a number between 0-9 which is used to assess strength of company’s financial position. 
	The Score is used by financial investors in order to find the best value stocks.


2. Operating process

	1) Scraps financial statements on the UK Yahoo finance website.
 	2) Decode financial data that I need in HTML
	3) Calcultate finance value and sum up of all achieved points gives Piotroski F-Score.

3. How to use it!

	1) Visit UK Yahoo financial website. (https://uk.finance.yahoo.com/)
	2) Search for the stock you want to know.
	3) Click 'Financials'
	Example page : https://uk.finance.yahoo.com/quote/AAPL/financials?p=AAPL
	4) Check final score (nine being the best).
